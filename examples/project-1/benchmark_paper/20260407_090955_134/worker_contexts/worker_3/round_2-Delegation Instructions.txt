## Task: Generate instances for OperationShape

**Target Shape:** `:OperationShape`
**Target Class:** `:Operation`
**Goal:** CREATE at least 1 instance of `:Operation` — none exist yet.

### Context
MyGlassFactory runs a glass production operation. An operation deploys systems and implements a process, is located at an operation location, and has a status. Existing resources to reference:
- Location: `ex:location/myGlassFactory`
- Process: `ex:process/glassProduction` (main), `ex:process/melting`, `ex:process/forming`
- Systems: `ex:installation/furnace`, `ex:installation/glass-former`, `ex:emissionpoint/stack1`, etc.

### Required Properties (minCount ≥ 1)
- `rdfs:label` (at least 1, plain literal)
- `dc:created` (xsd:dateTime, exactly 1)
- `dc:modified` (xsd:dateTime, exactly 1)
- `dc:issued` (xsd:date, exactly 1) — validity start
- `adms:status` → `adms:Status` (exactly 1) — use `concept:active`
- `ssn:deployedOnPlatform` → `:OperationLocation` (exactly 1) — use `ex:location/myGlassFactory`
- `ssn:implements` → `:Process` (exactly 1) — use `ex:process/glassProduction`

### Optional Properties
- `ssn:deployedSystem` → `ssn:System` — link to deployed systems
- `adms:identifier` → `adms:Identifier`
- `dc:valid` (xsd:date)
- `org:hasMember` → `:Contactpersoon`

```turtle
    rdfs:label "MyGlassFactory Operation 2024"@en ;

    ssn:deployedSystem ex:installation/furnace ;
    ssn:deployedSystem ex:installation/glass-former ;

    ssn:deployedSystem ex:emissionpoint/discharge1 .
```

### Instructions
1. Add `a :Operation` to each instance.
2. Use IRIs under `ex:operation/` namespace.
3. Link to `ex:location/myGlassFactory` via `ssn:deployedOnPlatform`.
4. Link to `ex:process/glassProduction` via `ssn:implements`.
5. Optionally add `ssn:hasDeployment` back-references on existing installations/systems (e.g., `ex:installation/furnace ssn:hasDeployment ex:operation/myGlassFactory-v1`).
