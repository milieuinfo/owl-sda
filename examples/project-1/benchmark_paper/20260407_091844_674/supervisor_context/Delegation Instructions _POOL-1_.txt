**Target Shape:** `:AbstractEmissionPointShape`

**Goal:** CREATE at least 1 instance of `:AbstractEmissionPoint` — none exist yet.

### Context
MyGlassFactory uses abstract emission points to represent conceptual or grouped emission release locations — distinct from concrete `:EmissionPoint` systems. An abstract emission point groups or represents emission points at a higher level (e.g., "all stack emissions", "all water discharges").

This shape has **no additional property constraints** beyond the type — any typed node conforms. However, add meaningful properties for data quality.
