## Task: Generate instances for ProcessVariableShape

**Target Shape:** `:ProcessVariableShape`
**Target Class:** `:ProcessVariable`
**Goal:** CREATE at least 2–3 instances of `:ProcessVariable` — none exist yet.

### Context
MyGlassFactory has glass production processes. Process variables represent input/output substances or quantities in a process (e.g., sand as input to the melting process, molten glass as output, natural gas as fuel input).

Existing processes to link to:
- `ex:process/glassProduction` — top-level
- `ex:process/melting` — subprocess (step of glassProduction)
- `ex:process/forming` — subprocess (step of glassProduction)

### Example Instances
```turtle
ex:processvariable/sand a :ProcessVariable ;
    rdfs:label "Sand (Input)"@en ;
    dc:type concept:rawMaterial ;
    pplan:isInputVarOf ex:process/melting .

ex:processvariable/naturalGas a :ProcessVariable ;
    rdfs:label "Natural Gas (Fuel Input)"@en ;
    dc:type concept:fuel ;
    qudt:hasUnit unit:M3 ;
    qudt:numericValue "1500.0"^^xsd:decimal ;
    pplan:isInputVarOf ex:process/melting .

ex:processvariable/moltenGlass a :ProcessVariable ;
    rdfs:label "Molten Glass (Output)"@en ;
    dc:type concept:product .
```

Also add `pplan:hasOutputVar ex:processvariable/moltenGlass` on `ex:process/melting`.
